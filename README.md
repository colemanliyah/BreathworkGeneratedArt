# BreathworkGeneratedArt

**Authors:** Liyah Coleman and Brian Bishop  

---

## Inspiration
The project was inspired by the desire to **detect inhale and exhale** and capture a **full breath cycle**. By analyzing the difference between pitch and roll from IMU data, we can track breathing patterns and use them to generate dynamic, real-time visual art.

---

## Features
- Real-time detection of **inhale** and **exhale**.
- Calculates a **full breath cycle**.
- Generates visual feedback or outputs based on breathing patterns.
